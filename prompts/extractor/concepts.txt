## 提取指导

- **论文引入的概念**：优先提取论文自己定义或提出的新概念，其次是论文重点使用的已有概念。
- **精确定义**：使用论文中的原始定义，不要改写。如果论文未给出正式定义，标注"论文未给出正式定义"。
- **作用说明**：解释该概念在论文方法/框架中的具体作用。
- **数量**：5-10 个核心概念。

以 JSON 格式输出：
```json
{
  "concepts": [
    {
      "name": "概念名称（英文原文）",
      "definition": "概念定义（使用论文原始表述）",
      "role": "在论文方法/框架中的具体作用"
    }
  ]
}
```
