## 提取指导

- **显式 vs 隐式**：先提取论文自述的局限性（"limitation"部分或讨论中的自我反思），再识别隐含的局限。
- **类型分类**：区分方法局限（方法本身的缺陷）、实验局限（数据集/规模限制）、假设局限（前提条件）、适用范围局限（领域限制）。
- **缺失信息**：如果论文未讨论局限性，基于方法本身的固有限制提取，但标注"论文未明确讨论"。

以 JSON 格式输出：
```json
{
  "limitations": [
    "局限性1（含类型说明）",
    "局限性2"
  ]
}
```
